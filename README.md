# opencode-quotes-plugin

A motivational quotes plugin for OpenCode.

## Installation

Run:
```bash
opencode plugin opencode-quotes-plugin -g
```

This will install the plugin globally.

## Usage

Once installed, the plugin replaces the default OpenCode tips with a rotating set of motivational quotes. Quotes are displayed at the bottom of the home screen with word wrapping and centered layout.

All commands are available from the command palette on the home screen:

- **Show tips** / **Hide tips** - Toggle quote visibility
- **Set quote source** - Choose quote sources: Built-in, Custom, or Both (default: Both)
- **Add quote** - Add a custom quote using the format: `"Your quote here" - Author Name`
- **Remove quote** - Search and remove a custom quote

## Quote Requirements

The list of quotes is mostly complete; the original list was trimmed down from over 300+ quotes. If you feel that a specific quote is unique, valuable, but missing, feel free to open a pull request to add it.

Quote Requirements:

- No persons who have built/achieved nothing
  - No "nepo babies"
  - No influencers
  - No motivational speakers
  - No comedians
  - No preachers
  - No "figurehead" leaders
  - No contemporary politicans
  - No contemporary philosophers
- No morally bankrupt individuals
  - No frauds, conmen
  - No hypocrites
  - No pseudo-scientists
  - No individuals advocating for radical individualism at the expense of the collective good
  - No individuals encouraging exploitation of others
  - No individuals generally regarded as immoral by historical or contemporary standards
- In general, avoid contemporary individuals

- Allowed:
  - Philosophers whose ideas may have influenced or been misapplied by immoral actors
  - Historical figures who in retrospect engaged in controversial activites that were considered normal within their era
